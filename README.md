# Offline-mental-health-chatbot
Offline NLP chatbot for mental health support built with python
Final year AI project - A Python based chatbot that works completely offline

## Features
- NLP based response system
- Works without internet
- Optimized from 8s to 1.5s response time
- Built with Python, NLTK, Tkinter

## How to run
1. Install requirements: pip install -r requirements.txt
2. Run: python app.py
